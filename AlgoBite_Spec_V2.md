# AlgoBite アプリケーション仕様書 (最新版)

## 1. アプリ概要
**AlgoBite** は、アルゴリズムとデータ構造を直感的に学習するためのインタラクティブなiOSアプリケーションです。「穴埋め形式のコードパズル」と「並べ替えクイズ」を通して、毎日少しずつ（Bite-size）プログラミングの基礎とアルゴリズムを学ぶことができます。クリア後には豊富なアルゴリズムアニメーションが再生され、視覚的な理解を助けます。

---

## 2. アーキテクチャと基本構造
- **フレームワーク**: SwiftUI を用いた MVVM (Model-View-ViewModel) アーキテクチャ。
- **エントリポイント**: `AlgoBiteApp.swift`
- **ルーティング**: `ContentView.swift` 内の `NavigationStack` による画面遷移。
  - 主な画面: `.problem` (パズル), `.reorder` (並べ替え), `.review` (復習), `.achievements` (実績), `.settings` (設定) など。
- **状態管理**: 
  - `GameViewModel.swift`: 毎日のチャレンジ（パズル/並べ替え）の進行状態、ストリーク、ヒント、解答判定を管理するメインのViewModel。
  - `StatsStore` / `BadgeStore` / `SettingsStore`: 統計情報、バッジ獲得状況、アプリ設定を管理 (`AppExtras.swift` 内)。
- **データ共有**: `UserDefaults` を用いた App Group (`group.app.Goto.Sakana.AlgoBite`) 経由で、ウィジェット (`AlgoBiteWidget`) と学習ストリークやクリア状態を共有。

---

## 3. 主要なデータモデル (`Models.swift` / `AppTypes.swift`)
- **`PuzzleProblem`**: 穴埋めパズルの問題データ。
  - ID, タイトル, 難易度 (Easy/Medium/Hard), トピック, 問題文, 穴埋め用のコードテンプレート (`{{slot_id}}` で定義), 選択肢(`PuzzleSlot`) を保持。
- **`ReorderQuiz`**: 並べ替えクイズのデータ。
  - 正解の順序配列(`answer`)と、ドラッグ用のプール配列(`pool`)を保持。
- **`DailyChallenge`**: その日のタスク。「パズル」と「並べ替え」を 3:1 の割合で出題するラッパー。
- **`SlotCheckState` / `ResultMood`**: 解答の正誤状態や、それに連動して変化する背景グラデーションの色状態（Neutral / Success / Fail）を定義。

## 4. コア機能
### 4.1. ホーム画面と毎日のチャレンジ
- 毎日のタスク（Today's Challenge）を提示。
- **ストリーク（連続学習記録）機能**: ロールケーキの上にイチゴが乗っていくカスタムアニメーション (`RollCakeStreak`) で視覚化。

### 4.2. 穴埋めパズル機能 (`ContentView.swift`)
- コードの空欄（スロット）を選択し、提示された選択肢チップから正しいコードブロックを埋める。
- **3段階のヒントシステム**: (1) 優しいテキスト説明 → (2) 1箇所だけ正解を埋める → (3) 全て埋めて自動判定 (`HintStore`)。

### 4.3. 並べ替えクイズ (`ReorderQuiz.swift`)
- **LCS (最長共通部分列) アルゴリズムを用いた独自判定**: 解答を提出した際、相対的な順序が正しいブロックはその場に残り、間違っている（LCSに含まれない）ブロックのみが揺れてプールに戻る仕組みを実装。

### 4.4. 復習と実績 (`AppExtras.swift`)
- **復習モード (`PracticeSession`)**: 過去の問題をストリークに影響せず解き直せる機能。
- **バッジと統計**: クリア数や連続日数に応じたバッジアンロック機能（紙吹雪と絵文字の豪華なアニメーション演出付き）と、28日間の活動ヒートマップ。

## 5. 収録問題とアルゴリズムアニメーション
合計 **100問** のコードパズルと **29種類** の並べ替えクイズが実装されています。

### 問題の難易度とトピック内訳 (全100問)
- **難易度**: Easy 約35問, Medium 約50問, Hard 約15問
- **主なトピック**:
  - 動的計画法 (DP): 18問
  - 木構造 (Tree): 14問
  - ソート (Sorting): 8問
  - グラフ (Graph / BFS / DFS): 9問
  - 文字列 (String): 5問
  - 連結リスト (Linked List): 5問
  - バックトラッキング: 4問
  - その他、スライディングウィンドウ、Two Pointers、Trie木、ビット演算など。

### アニメーション・システム (`TopicAnimations.swift` / `Animations_*.swift`)
問題クリア後に表示される `ExplanationView` では、アルゴリズムの挙動を示す **105種類以上** のインタラクティブな SwiftUI アニメーションが用意されています。
- **制御方式**: `@State token` と `DispatchQueue.main.asyncAfter` を用いた、キャンセル・再実行可能なステップ実行アニメーション。
- **カテゴリ**: DPの表埋め、グラフ探索（BFS/DFS/Dijkstra）、ソートのステップ実行、ポインタの移動、再帰ツリーの展開などを網羅。

---

## 6. デザインシステム (`PopStyle.swift`)
Duolingoにインスパイアされた、親しみやすくポップなUIデザイン（Pop Aesthetic）を採用。
- **カラーパレット (`Pop`)**: `primary` (グリーン), `accent` (アンバー), `danger` (レッド) を基調とし、ライト/ダークモードに動的対応。
- **立体的なコンポーネント**: 押下時に沈み込む 3D ボタン (`PopButton`) や、角丸とボーダーを備えたカード (`PopCard`)。
- **カスタムイラストレーション**: 画像アセット (`Assets.xcassets`) を使わず、すべて SwiftUI の `Shape` や `Path` を使って描画されたスイーツ（クッキー、ドーナツ、ケーキ）や、トピックごとの抽象アルゴリズムアート (`TopicIllustration`)。

---

## 7. ウィジェットと開発ツール
- **AlgoBiteWidget**: iOSホーム画面用のウィジェット。現在のストリーク数と当日のクリア状況を一目で確認可能。
- **DebugCapture (`DebugCapture.swift`)**: 開発時の自動テストやスクリーンショット撮影を補助するユーティリティ。特定の画面への直接遷移や、正解/不正解の自動入力 (`-autoplay`) をサポート。
